🎙️ DiktatApp
Automatisches Diktat für die Schule – direkt im Browser, installierbar auf iPhone & Android.
Was die App kann
📋 Text einfügen – beliebigen Diktat-Text eintippen oder einfügen
📷 Foto aufnehmen – Diktat abfotografieren, Text wird automatisch erkannt
🎙️ Vorlesen – jeden Satz bis zu 4× wiederholen, Satzzeichen werden mitdiktiert
⚙️ Einstellbar – Tempo, Wiederholungen und Schreibpause frei wählbar
📵 Offline-fähig – funktioniert nach dem ersten Laden auch ohne Internet
App installieren
iPhone / iPad: Link in Safari öffnen → Teilen (□↑) → „Zum Home-Bildschirm"
Android: Link in Chrome öffnen → Menü ⋮ → „App installieren"
Foto-Erkennung einrichten
Für die Foto→Text-Funktion wird ein kostenloser Anthropic API-Schlüssel benötigt:
Account erstellen auf console.anthropic.com
Unter „API Keys" einen neuen Schlüssel erstellen
In der App unter ⚙️ Einstellungen eintragen
Der Schlüssel wird ausschließlich lokal auf dem Gerät gespeichert.
Technologie
Reine Progressive Web App (PWA) – kein Framework, kein Build-Tool, kein App Store.
Text-zu-Sprache via Web Speech API. OCR via Claude Vision API (Anthropic).
Kostenlos. Werbefrei. Open Source.
